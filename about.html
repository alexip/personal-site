About

A quiet moment shows one affirming line and one journal prompt each day. The pair rotates at your local midnight.

The idea is small on purpose: a single sentence and a single question is usually enough — to interrupt a thought, soften a mood, or start a journal entry. No accounts, no analytics, no sign-up forms. Just the page.

Made by Alex, somewhere in the Bay Area. The code is on GitHub if you're curious how it works.
